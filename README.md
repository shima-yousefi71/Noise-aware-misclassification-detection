# Noise-Aware Misclassification Attack Detection in Collaborative DNN Inference

## Overview
This repository contains the code for our work on detecting adversarial misclassification attacks in collaborative DNN inference under noisy conditions.
Accepted at IEEE/ACM CCGrid 2026
## Repository Structure (Planned)
- `src/` : source code
- `data/` : datasets or processed features
- `models/` : saved models
- `results/` : outputs and evaluation results
- `notebooks/` : experiments and analysis notebooks

This repository is under development
